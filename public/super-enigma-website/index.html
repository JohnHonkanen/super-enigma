  • Say hello to "Super-Enigma".

    Super-Enigma is a real-time strategy (RTS) game of diplomacy, conflict and conquest for two or more players.

Learn more

Our Latest Project.

The standard version of Super-Enigma is played in 2D map broken up into 20 territories. Each territory produces a unit per game tick and each territory can be instructed to move its units to another territory provided the destination can be reached by drawing a straight line from the starting territory to the end destination.

The goal of the game is to take over all the opposing players territories. The more territories the player controls, the more units they can produce per game tick. Territories can also provide a boost in unit traveling speed and production, which will be vital for each player to secure to ensure their victory.

Territories are won and held by the player who have 1 or more units. Territories unit and point production can be boosted by the player using temporary skill boosts in exchange for their points until the territory is lost.

Installation Steps:

  • Download the Windows installer from the Nodes.js® web site.
  • Run the installer (the .msi file you downloaded in the previous step.)
  • Follow the prompts in the installer (Accept the license agreement, click the NEXT button a bunch of times and accept the default installation settings).
  • Restart your computer. You won’t be able to run Node.js® until you restart your computer.

Running NodeJS:

  • Open your terminal and navigate to the root folder, where server.js is located.
  • Install Dependencies via npm install.
  • Run Node server via npm start, or node server.js in the root folder.
